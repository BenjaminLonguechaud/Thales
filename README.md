# Thales Projects

This repository contains several sub-projects related to Thales CipherTrust and tokenization solutions. Each sub-project demonstrates different integration or usage scenarios for Thales security products.

## Sub-Projects

### 1. CADP
Application Level Encryption with Thales CipherTrust Application Data Protection. Includes prerequisites and deployment instructions for CipherTrust Manager.

### 2. CTVL_Java
Sample code and integrations for CipherTrust Vaultless Tokenization (CT-VL) in Java. Includes setup instructions and prerequisites for using the org.json package and configuring deployment variables.

### 3. CTVL_Net
A .NET example application leveraging the Tokenization REST API to tokenize and detokenize credit card data. Includes links to Thales documentation and a quick installation guide for the virtual Tokenization Server.

### 4. CTVL_Python
A Python-based GUI application for managing tokenization and detokenization of sensitive data. Features a multi-tab interface, user authentication, configurable server URL, and secure data handling. See the subproject's README for installation and usage details.

### 5. extract_keys
A simple Python script for extracting keys from Thales CipherTrust. Easily customizable for different input/output behaviors. See the subproject's README for usage instructions.

### 6. Database/Postgres
A simple Python application to connect to and interact with a PostgreSQL database deployed in a Kubernetes cluster.

For more details on each sub-project, refer to their respective README files.
